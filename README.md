## Requirements

- Anaconda3
- Python 3.8
- CUDA Toolkit 11.1
- cuDNN 8
- NVIDIA GPU Driver 460.x

## Setup

```sh
conda env create -f=env_name.yml
activate py38
pip install -r requirements.txt
```
